# Portfólio — Edson F. Lima

Portfólio pessoal desenvolvido em HTML, CSS e JavaScript puro, sem frameworks ou dependências externas além das fontes do Google.

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, media queries)
- JavaScript vanilla (IntersectionObserver, scroll spy, tema)
- Google Fonts — DM Serif Display, DM Mono, Outfit

## ✨ Funcionalidades

- Modo escuro nativo com alternância para modo claro (salvo no localStorage)
- Navegação lateral com scroll spy (ativa o link conforme a seção visível)
- Animações de entrada ao scroll (reveal + barras de habilidade)
- Layout responsivo — desktop, tablet e mobile
- Menu hambúrguer no mobile
- Formulário de contato com validação básica
